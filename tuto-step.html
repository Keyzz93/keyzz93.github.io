Tutoriel complet : créer un site web

Étape 1 — L’idée et le plan

Avant de coder, réfléchis à ce que tu veux faire. Définis le but, les pages et la structure.

Étape 2 — HTML : structure

Le HTML organise ton site : titres, paragraphes, images, liens.

<header>Menu</header>
<main>Contenu</main>
<footer>Bas de page</footer>

Étape 3 — CSS : le style

Le CSS rend ton site beau : couleurs, polices, marges, animations.

body { background: #06060a; color: white; font-family: 'Poppins', sans-serif; }
header { padding: 20px; background: rgba(255,255,255,0.05); }

Étape 4 — JavaScript : interactions

Animations, clics et formulaires sont gérés avec JS.

document.querySelector('.btn').addEventListener('click', function() {
  alert('Bonjour !');
});

Étape 5 — Formulaire de contact

Permet aux visiteurs de t’envoyer un message.

<form action="mailto:tonemail@gmail.com" method="post">
  <input type="text">
  <input type="email">
  <textarea></textarea>
  <button>Envoyer</button>
</form>

Étape 6 — Mise en ligne

Publie ton site sur GitHub Pages, Netlify ou un hébergeur payant.
