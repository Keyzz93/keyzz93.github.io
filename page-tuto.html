Comment créer un site web facilement ?

Voici les étapes simples :

  • 1. Va sur GitHub → crée un repository
  • 2. Nom du repo : tonpseudo.github.io
  • 3. Ajoute un fichier index.html
  • 4. Active GitHub Pages dans Settings → Pages
  • 5. Ton site est en ligne gratuitement
