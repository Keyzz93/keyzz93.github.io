Étape 2 — HTML : la structure

Le HTML définit le squelette de la page :

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Mon site</title>
</head>
<body>
  <header></header>
  <main></main>
  <footer></footer>
</body>
</html>
  • <head> : titre, meta, CSS
  • <body> : contenu visible
  • Balises sémantiques : <nav>, <section>, <article>, <footer>

Exercice : crée un fichier index.html avec header, section et footer.
