Comment créer un site web – Tutoriel simple et complet

Réalisé par Haytam – Dédié à Madame Duchâteau

Étape 1 : Créer un dossier pour ton site

Avant de commencer, tu dois créer un endroit où tu vas mettre tous les fichiers de ton site. Voici comment faire :

  • Va sur ton Bureau.
  • Fais un clic droit → Nouveau → Dossier.
  • Donne un nom au dossier, par exemple : MonSite.
  • Ce dossier contiendra toutes les pages et images de ton site.

Étape 2 : Créer la page principale (index.html)

Maintenant que ton dossier est prêt, tu vas créer ton premier fichier. C’est la page la plus importante : index.html.

  • Ouvre le dossier MonSite.
  • Fais un clic droit → Nouveau → Document texte.
  • Renomme-le en index.html. (IMPORTANT : enlève le .txt)
  • Ouvre-le avec Bloc-notes ou VS Code si tu en as un.

Tu peux maintenant y mettre du code HTML (comme celui que je t'ai donné pour ton site).

Étape 3 : Comprendre ce qu’est HTML

HTML est un langage utilisé pour construire une page web. Il sert à placer le texte, les titres, les images, les boutons… C’est comme la structure d’une maison.

Exemple d'une phrase en HTML :

<h1>Bienvenue sur mon site</h1>

Étape 4 : Ajouter du style avec un fichier CSS

Le CSS sert à rendre ton site beau : couleurs, ombres, fonds, tailles, etc. C’est la décoration de la maison.

  • Dans ton dossier MonSite, crée un nouveau fichier.
  • Appelle-le : style.css.
  • Tu mettras dedans toutes les couleurs et le style.

Étape 5 : Ouvrir ton site pour le voir

Quand tes fichiers sont prêts :

  • Double-clique simplement sur le fichier index.html.
  • Ton navigateur (Chrome / Edge) ouvrira ton site.
  • Tu peux le modifier et recharger la page pour voir les changements.

Étape 6 : Mettre ton site en ligne (GitHub Pages)

Pour rendre ton site accessible à tout le monde :

  1. Crée un compte sur GitHub.
  2. Crée un nouveau dépôt (New Repository).
  3. Mets tes fichiers dedans (index.html et style.css).
  4. Va dans Settings → Pages.
  5. Choisis “Deploy from branch”.
  6. Ton site sera publié quelques minutes après.

Étape 7 : Test final

Quand tout marche, tu peux donner ton lien (ex : tonpseudo.github.io) pour que les gens voient ton site.
