Tutoriel complet : comment créer un site web

Voici toutes les étapes pour comprendre comment un site est créé. Tu vas voir : c’est simple quand c’est bien expliqué.

Étape 1 — Comprendre les trois langages du web

Avant de créer ton site, tu dois comprendre les 3 langages utilisés partout :

  • HTML → le squelette du site (titre, texte, boutons…)
  • CSS → le style (couleurs, centré, tailles…)
  • JavaScript → les animations & interactions

Étape 2 — Créer les fichiers du site

Dans ton dossier, crée trois fichiers :

  • index.html
  • style.css
  • script.js

Étape 3 — Remplir le HTML

Le HTML sert juste à afficher du contenu. Par exemple : 

<h1>Bienvenue</h1>
<p>Ceci est mon site.</p>

Étape 4 — Ajouter du style avec CSS

Le CSS rend ton site beau :

body {
  background: black;
  color: white;
  font-family: Arial;
}

Étape 5 — Ajouter des animations avec JavaScript

JavaScript permet d’ajouter des effets comme le fond animé de ce site : 

console.log("Bienvenue sur mon site !");

Étape 6 — Mettre ton site en ligne

Tu peux utiliser GitHub Pages : gratuit, rapide et simple. Il suffit d’envoyer tes fichiers et d’activer “Pages”.
