Étape 3 — CSS : le style

Le CSS permet d’embellir ton site :

body { font-family: Arial, sans-serif; background: #0a0a0a; color: white; }
header { padding: 20px; }
.card { background: rgba(255,255,255,0.05); padding: 20px; border-radius: 10px; }
  • Palette de 2–3 couleurs principales
  • Espacement (padding/margin) pour lisibilité
  • Polices : Google Fonts

Exercice : crée style.css et applique un fond sombre, police et cartes arrondies.
